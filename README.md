**1) What is the difference between null and undefined?**

- **null:** null ইউজার নিজে থেকে ডিফাইন করে দেয় ।
- **undefined:** কোন একটা vaiable declare করেছি, কিন্তু সেইটার মান এখনো ইনিশিয়ালাইজ করিনি তখন সেই ভ্যালুটার মান হয় undefined

###

**2) What is the use of the map() function in JavaScript? How is it different from forEach()?**

- JavaScript-এ `map()` function একটা array এর মধ্যে নির্দিষ্ট কাজ করার পর নতুন একটা array return করে।
- forEach() array এর মধ্যে নির্দিষ্ট কাজ করার পর কোন কিছু return করে নাহ । এইটাই `map()` এবং `forEach()` এর মধ্যে মুল পার্থক্য ।

###

**3) What is the difference between == and ===?**

- `==` মূলত value check করে ।
- `===` value + data-type দুইটাই check করে

**4) What is the significance of async/await in fetching API data?**

- কোন API থেকে data synchronize way তে load করার জন্য async/await ব্যবহৃত হয় ।

###

**5) Explain the concept of Scope in JavaScript (Global, Function, Block).**

- Global: গ্লোবাল scope javascript এ যেকোনো জাইগা থেকে ব্যবহার করা যায় ।
- Function: function scope শুধুমাত্র function ব্লক এর মধ্য থেকেই ব্যবহার করা যায় ।
- Block: ব্লক scope শুধুমাত্র নির্দিষ্ট ব্লক এর মধ্য থেকেই ব্যবহার করা যায় ।
